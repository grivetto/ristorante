# The Gastronomic Hub - Beta 0.0.1

Benvenuti in **The Gastronomic Hub**, un'esperienza web immersiva che celebra la diversità culinaria attraverso il design.

## Le 4 Anime del Gusto

Il sito è strutturato come un ecosistema dinamico che cambia pelle a seconda della tua scelta:

1.  **Pizzeria (Popolare/Vibrante)**: Colori caldi, tipografia bold e un layout energico che richiama la convivialità della pizza appena sfornata.
2.  **Tavola Calda (Pratico/Veloce)**: Design pulito, scattante e funzionale. Ideale per chi cerca efficienza senza rinunciare alla qualità.
3.  **Ristorante Standard (Accogliente/Familiare)**: Toni della terra, font graziati e un'atmosfera che sa di casa e tradizione.
4.  **Ristorante di Lusso (Elegante/Minimalista)**: Nero profondo, oro, ampi spazi bianchi e una tipografia sottile per un'esperienza d'élite.

## Come Navigare

-   **Landing Page**: All'arrivo, verrai accolto da un portale di scelta asimmetrico. Clicca su una delle quattro sezioni per attivare lo "Style Switcher".
-   **Style Switcher**: Una volta entrato in una sezione, l'intera palette cromatica, i font e le animazioni si adatteranno istantaneamente al tema scelto.
-   **Ritorno**: Puoi sempre tornare all'hub centrale tramite il logo o il menu di navigazione.

## Dettagli Tecnici

-   **Framework**: React 19 + Vite
-   **Styling**: Tailwind CSS 4 con variabili CSS dinamiche.
-   **Animazioni**: Framer Motion per transizioni fluide tra i temi.
-   **Deployment**: Ottimizzato per GitHub Pages (percorsi relativi).
